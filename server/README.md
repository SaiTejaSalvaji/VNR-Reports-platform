# Institute Reports - Backend Server

A backend server for managing institute reports and data.

## 📁 Project Structure

- **Database schemas**: Located in `/sql` directory
- **Configuration**: Environment variables in `src/configs/env.ts`

## 🚀 Getting Started

1. Configure your environment variables in `src/configs/env.ts`
2. Set up your database using the schemas in `/sql`
3. Install dependencies and start the server

## 📊 Database

For database schemas and structure, refer to the `/sql` directory.
