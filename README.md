 # Institute Monthly Report Portal

## Features

- Simplified workflow for submitting and managing monthly reports
- Secure authentication with role-based access controls
- Interactive dashboard for monitoring report status and analytics
- Export capabilities supporting various formats

## Getting Started

1. **Clone the repository**
2. **Install dependencies**
    ```bash
    npm install
    ```
3. **Launch the development server**
    ```bash
    npm start
    ```

## Usage

- Sign in with your institute credentials
- Create, edit, and submit monthly reports
- Access previously submitted reports and review analytics
